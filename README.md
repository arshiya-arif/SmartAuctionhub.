Smart AI-Powered AuctionHub is a fully responsive web-based auction platform that leverages Artificial Intelligence to make online bidding faster, smarter, and more secure. Built with React.js, Node.js, and AI/ML models, it offers real-time bidding, automated bidding up to a set limit, and bid price prediction.

**1.Clone Repository**
```bash
git clone https://github.com/arshiya-arif/SmartAuctionhub..git
cd SmartAuctionhub
```
**2.Install Dependencies**

Frontend
```
cd frontend
npm install
```
Backend
```
cd backend
npm install
```

**3.Set Environment Variables**

In .env file


**4.Run Application**

Backend
```
cd backend
npm run dev
```
Frontend
```
cd frontend
npm run dev
```
